Docker Tag е‘ҪеҗҚ

дёҖиҲ¬дҫҶи¬ӣпјҢеҗҢдёҖеҖӢdocker imageжңғжҸҗдҫӣеӨҡеҖӢдёҚеҗҢзҡ„зүҲжң¬пјҢжҜҸеҖӢзүҲжң¬жңғйҷ„дәҲдёҚеҗҢзҡ„tagпјҢд»ҘдҪңжЁҷиӯҳгҖӮдҪҶд»Ҙdocker imageзҡ„з¶ӯиӯ·иҖ…дҫҶи¬ӣпјҢе®ғзҡ„tagйҖҡеёёд»ЈиЎЁзҡ„жҳҜиҮӘе·ұзЁӢејҸзҡ„зүҲжң¬иҷҹгҖӮдёҚйҒҺйҖҷеҖӢзүҲжң¬иҷҹеҚ»еӯҳеңЁеҫҲеӨҡи®Ҡж•ёпјҢе°ұи®“зӯҶиҖ…еҘҪеҘҪең°йҖҗдёҖиӘӘжҳҺгҖӮ

зЁӢејҸзҡ„зүҲжң¬иҷҹ

еңЁжІ’жңүDockerзҡ„е№ҙд»ЈпјҢе…¶еҜҰжүҖжңүи»ҹд»¶еңЁзҷјдҪҲжҷӮпјҢйғҪжңғжЁҷзӨәзүҲжң¬иҷҹпјҢж–№дҫҝдҪҝз”Ёж–№жҳҺзўәиҝҪи№Өе•ҸйЎҢпјҢиҮӘиЎҢйҒёж“ҮеҚҮзҙҡгҖҒйҷҚзҙҡд»Ҙи§Јжұәзӣёе®№жҖ§е•ҸйЎҢгҖӮеӨ§е®¶иҰҒйҮҚзҸҫе•ҸйЎҢпјҢд№ҹиғҪжё…жё…жҘҡең°йҮҚзҸҫгҖӮжүҖд»Ҙdocker imageзҡ„tagпјҢеңЁжҹҗзЁӢеәҰпјҢйғҪжҳҜд»ЈиЎЁзҷјдҪҲиҮӘе·ұзҡ„зЁӢејҸзүҲжң¬иҷҹгҖӮдҪҶд»ҘеүҚзҡ„е№ҙд»ЈпјҢи»ҹд»¶еә•еұӨзҡ„дҫқиіҙпјҢдҫӢеҰӮOSеұӨйқўзҡ„е…ұдә«зЁӢејҸеә«пјҢеүҮдёҚеңЁзҷјдҪҲзҡ„з®ЎжҺ§дёӯпјҢжүҖд»ҘйҒҺеҺ»зҡ„зЁӢејҸпјҢеңЁи·Ёйӣ»и…Ұе®үиЈқжҷӮпјҢйғҪжңғеҮәзҸҫзјәе°‘жҹҗдәӣе…ұдә«еә«зҡ„е•ҸйЎҢгҖӮиҖҢдҪҝз”ЁдәҶDockerеҫҢпјҢimageд»Ҙе…§зҡ„е…ұдә«еә«зҡ„йғҪжңғеңЁжү“еҢ…зҡ„йӮЈдёҖеҲ»еӣәе®ҡе’ҢзҷјдҪҲпјҢе°ұдёҚжңғжңүжјҸзҡ„е•ҸйЎҢгҖӮ

еә«жӣҙж–°пјҢжҖҺйәјиҫҰ

дёҠйқўиӘӘеҲ°imageеҸҜд»Ҙжү“еҢ…е…ұдә«еә«пјҢдҪҶе•ҸйЎҢжҳҜе…ұдә«еә«д№ҹжңғжңүе®үе…ЁжҖ§жӣҙж–°е•ҸйЎҢпјҢйӮЈйәје°Қdocker imageзҡ„з¶ӯиӯ·иҖ…дҫҶи¬ӣпјҢе®ғиҮӘе·ұзҡ„tagеҸҲи©ІеҰӮдҪ•е‘ҪеҗҚпјҹ

еӣ зӮәеә«зҡ„йҮҸеҸҜеӨ§еҸҜе°‘пјҢжүҖд»ҘдёҖиҲ¬дҫҶиӘӘпјҢйғҪдёҚеҸҜиғҪе®Ңе…ЁжҠҠеҗ„еҖӢеә«зҡ„зүҲжң¬иҷҹеҜ«еңЁиҮӘе·ұзҡ„tagдёҠгҖӮйҖҖиҖҢжұӮе…¶ж¬ЎпјҢе°ұжҳҜз”Ё"зүҲжң¬иҷҹ+ж—Ҙжңҹ"пјҢеә«зҡ„зҙ°зүҲжң¬иҷҹпјҢе°ұеӯҳеңЁеҺҹе§Ӣзўјз•¶дёӯгҖӮUbuntu е°ұжҳҜйҖҷжЁЈзҡ„дҫӢеӯҗгҖӮ

дёҚйҒҺ"зүҲжң¬иҷҹ+ж—Ҙжңҹ"зҡ„е‘ҪеҗҚж–№ејҸзңҹзҡ„ж–№дҫҝе—ҺпјҹжҜҸж¬ЎдёӢйҒҠз”ЁжҲ¶жғіжӣҙж–°еҺ»жңҖиҝ‘зүҲжң¬пјҢйғҪиҰҒиҮӘе·ұжүҫдёҖж¬ЎжңҖиҝ‘зҡ„ж—ҘжңҹгҖӮйҖҷжЁЈе°ҚеҫҲеӨҡз”ЁжҲ¶дҫҶи¬ӣйғҪдёҚеӨ ж–№дҫҝгҖӮжүҖд»ҘdockerеҸҲжҸҗдҫӣдәҶдёҖеҖӢйҮҚtagзҡ„еҠҹиғҪгҖӮдҫӢеҰӮubuntu:nobleпјҢеңЁж—©дәӣжҷӮеҖҷжҢҮи‘—noble-20240904.1пјҢз„¶еҫҢйҒҺе№ҫеӨ©пјҢеҸҲжҢҮеҗ‘жӣҙж–°зҡ„noble-20241009гҖӮжӣҙеёёиҰӢзҡ„жҳҜlatestпјҢжҜҸж¬ЎimageйғҪй җиЁӯжңғеӯҳеңЁпјҢdockerд№ҹеёҢжңӣеӨ§е®¶жңғе®ҡжңҹжӣҙж–°йҖҷеҖӢtagпјҢи®“еӨ§е®¶еҸҜд»Ҙжӣҙжҳ“ең°жүҫеҲ°жңҖж–°зүҲжң¬гҖӮ

иЁ»: йҖҷи·ҹgit tagжңүжүҖдёҚеҗҢпјҢgit tagдёҰдёҚй җжңҹжңғи®Ҡзҡ„гҖӮз•¶еҚ”дҪңиҖ…ж”¶еҲ°tagеҫҢпјҢйӮЈжҖ•дёҠйҒҠеҲ»ж„Ҹжӣҙж–°tagжҢҮйҮқпјҢеҚ”дҪңиҖ…жІ’жңүеҲӘйҷӨеҺҹtagд№ӢеүҚпјҢйғҪдёҚжңғзҹҘйҒ“tagжӣҙж–°еҺ»дәҶе“ӘиЈҸгҖӮ

жҲ‘еҖ‘и©ІеҰӮдҪ•йҒё

еңЁзҷјдҪҲж–№е’Ңеј•з”Ёж–№дҫҶи¬ӣпјҢеј•з”ЁжҷӮеҸҜд»ҘжҳҺзўәдҪҝз”Ёе”ҜдёҖзҡ„"зүҲжң¬иҷҹ+ж—Ҙжңҹ"пјҢе°Қз©©е®ҡжҖ§дҫҶи¬ӣжҳҜжңүж„Ҹзҫ©зҡ„гҖӮдёҚйҒҺеӨҡеӨҡе°‘е°‘пјҢжңғз”ўз”ҹйЎҚеӨ–зҡ„жҷӮй–“жҲҗжң¬гҖӮзҷјдҪҲж–№дҫҶиӘӘпјҢе°ұжҳҜеӨҡз”ЁдәҶдёҖдәӣе„Іеӯҳз©әй–“пјҢж–№дҫҝеј•з”Ёж–№еҸҜд»ҘйҡЁжҷӮжүҫеҲ°иҲҠ(еә«)зүҲжң¬гҖӮиҖҢеј•з”Ёж–№пјҢе°ұиҰҒжүӢеӢ•дҝ®ж”№еј•з”ЁиҷҹпјҢдҪңзӮәй©—ж”¶дҫқж“ҡпјҢиҮӘеӢ•жӣҙж–°зҡ„йӣЈеәҰжҜ”ијғеӨ§гҖӮ

дҪҶе°Қж–јиҮӘеӢ•жӣҙж–°иҰҒжұӮжҜ”ијғеӨ§зҡ„жғ…жіҒдёӢпјҢеҸҜиғҪе°ұжҳҜдҪҝз”ЁlatestжҲ–иҖ…жңғйҡЁжҷӮжӣҙж–°зҡ„share tag(е…ұз”Ёtag)жҜ”ијғеҜҰйҡӣгҖӮдҪҶжҲ‘еҖ‘д№ҹдҫқз„¶иҰҒе®ҡдёҖдәӣж–№ејҸеҺ»зүҲжң¬жӣҙж–°иЁҳйҢ„пјҢдҫӢеҰӮпјҡеҗҢжҷӮдҪҝз”Ё

Garbage collection on registry

config

storage:
  filesystem:
    rootdirectory: /var/lib/registry
  delete:
    enabled: true

command

bin/registry garbage-collect --delete-untagged=true /etc/docker/registry/config.yml

some image keep using same tag to keep check latest version; but older blob is not able to be deleted according to normal manifest garbage collection. Must use --delete-untagged flag to delete unused blob
